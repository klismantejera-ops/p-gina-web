Procesador (CPU)

El procesador es el cerebro del computador, ejecuta instrucciones y procesa datos para realizar todas las tareas.

Memoria RAM

La memoria RAM es una memoria temporal que guarda datos e instrucciones que la CPU necesita en tiempo real.

Disco Duro

Dispositivo de almacenamiento permanente donde se guardan programas, documentos y el sistema operativo.

Placa Base (Motherboard)

Placa que conecta todos los componentes del computador permitiendo su comunicación y funcionamiento conjunto.

Tarjeta Gráfica

Hardware especializado que procesa las imágenes y señales visuales enviadas al monitor para mostrar gráficos y videos.

Sistema Operativo

Software que administra y coordina los recursos del hardware y provee servicios a los programas y usuarios.
